#factorial function with recursion#

        
            const get_factorial = (num) => {
                if (num === 1) return 1;
                if (num === 0) return 0;
                else {
                    return num * get_factorial(num - 1);
                }
            }
code with JS
**위의 코드는 O(n)의 시간 복잡도를 가진다
**그러나 O(n2)으로도 구현 가능하다
**그러나 O(lognn)으로도 구현 가능할지는 모르겠다
